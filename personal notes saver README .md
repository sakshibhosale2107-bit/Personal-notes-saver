**Personal Notes Saver**

A simple CLI-based note-taking program created for the Viva/Project.

It lets the user add, view, and delete small text notes.





**How to Run**

1\. Make sure Python is installed on your system.

2\. Open any terminal (Command Prompt, PowerShell, or VS Code Terminal).

3\. Run the file:

&nbsp;  python notes.py





**Features**

* Add a new note
* View all saved notes
* Delete a note
* Stores notes in a text file (notes.txt)





**Files Included**

1. notes.py — main Python program
2. notes.txt — stores the saved notes
3. README.md — documentation file





**Requirements**

Python 3.x

(No external libraries required.)





**Usage Example**

After running the program, you will see a menu like:

1\. Add Note

2\. View Notes

3\. Delete Note

4\. Exit
